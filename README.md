# rs_protobuf
Rust google protocol buffer server, client example

---
## dependencies
`protobuf`

### macOS
```
$ brew install protobuf
```

### Ubuntu
```
$ apt install protobuf-compiler
```

### Rocky
```
$ dnf install protobuf-compiler
```
---

## build
```
$ cargo build
```

## run
### server
```
$ cargo run --bin server
```

### client
```
$ cargo run --bin client
```
